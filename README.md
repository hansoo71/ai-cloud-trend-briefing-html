# AI Cloud Trend Briefing HTML

- 최신 Pages: https://hansoo71.github.io/ai-cloud-trend-briefing-html/
- 날짜별 Pages: https://hansoo71.github.io/ai-cloud-trend-briefing-html/20260524/
- canonical source: `src/`
- Pages build: `docs/`
- source notes: `docs/source/`

## 실행/검증

```bash
python3 -m http.server -d docs 8080
```

브라우저 또는 HTTP 클라이언트로 `/`, `/20260524/`, 이미지 자산을 확인합니다.
