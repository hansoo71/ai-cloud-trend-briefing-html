# AI/클라우드/엔지니어링 일일 브리핑 HTML

공개 출처 기반 일일 기술 브리핑을 HTML과 Markdown으로 보관하는 저장소입니다.

- 최신 HTML: `docs/index.html`
- 날짜별 HTML: `docs/20260620/index.html`
- 원문 근거: `docs/source/20260620-source-notes.md`
- 텔레그램 요약: `docs/source/20260620-telegram.md`

## 실행
로컬 확인은 저장소 루트에서 `python3 -m http.server 8020 -d docs`로 수행합니다.
