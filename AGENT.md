# Agent Guide

- 원출처 우선: 공식 블로그, 벤더 공지, 원문 보도를 우선 인용한다.
- Google News RSS는 discovery layer로만 사용하고 최종 링크는 canonical URL로 정리한다.
- 접근 제한이 있으면 본문을 읽은 것처럼 쓰지 말고 제한을 명시한다.
- `src/` 수정 후 `docs/index.html`, `docs/YYYYMMDD/index.html`, standalone HTML을 함께 동기화한다.
- 이미지 자산은 `src/assets/images`, `docs/assets/images`, `docs/YYYYMMDD/images`에 복제한다.
