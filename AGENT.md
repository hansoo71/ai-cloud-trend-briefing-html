# 에이전트 편집 규칙

1. 공개 원문 URL과 발표일이 확인된 항목만 본문에 포함합니다.
2. 화면에 보이는 텍스트와 접근성 라벨은 한국어 존댓말로 유지합니다.
3. 제품명, 기업명, 공식 기능명은 원문 표기를 유지하되 필요한 경우 한국어 설명을 덧붙입니다.
4. `src/index.html`, `docs/index.html`, 날짜별 `docs/YYYYMMDD/index.html`, 단독 HTML을 항상 함께 동기화합니다.
5. 원문 전체 추출물이 저작권 또는 접근 제한 위험이 있으면 공개 저장소에 올리지 않고 요약 근거만 보존합니다.
